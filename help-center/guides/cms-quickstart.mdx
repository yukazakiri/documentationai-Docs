## Setup Koamishin CMS

<Steps>
  <Step title="Install package">
    ````bash
    composer require koamishin/cms
    ````
  </Step>
  <Step title="Publish assets">
    ````bash
    php artisan vendor:publish --tag=koamishin-cms
    ````
  </Step>
  <Step title="Run migrations">
    ````bash
    php artisan migrate
    ````
  </Step>
</Steps>

You're ready to start building with Koamishin CMS!
